METHOD 1: TEXTBLOB

Versatile Python library offering simple sentiment analysis, translation, and more. An accessible choice for text processing tasks in your analysis.

METHOD 2: VADER

VADER uses a combination of lexicon-based and grammatical rules to determine the sentiment polarity (positive, negative, or neutral) of a piece of text.

METHOD 3: BERT

"nlptown/bert-base-multilingual-uncased-sentiment." This is a bert-base-multilingual-uncased model finetuned for sentiment analysis on product reviews in: English, Dutch, German, French, Spanish, Italian.

METHOD 4: BERT

"finiteautomata/bertweet-base-sentiment-analysis." Model trained with SemEval 2017 corpus (around ~40k tweets). Base model is BERTweet, a RoBERTa model trained on English tweets.

METHOD 5: RoBERTa

"cardiffnlp/twitter-roberta-base-sentiment-latest. This is a RoBERTa-base model trained on ~124M tweets from January 2018 to December 2021, and finetuned for sentiment analysis with the TweetEval benchmark."
